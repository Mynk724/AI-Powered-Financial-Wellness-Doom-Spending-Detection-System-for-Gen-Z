Gen Z faces a unique financial crisis — not from lack of income, but from behavioral spending patterns amplified by one-tap payments, BNPL services, and algorithm-driven shopping apps. Existing budgeting tools track numbers; none of them track the psychology behind those numbers. This system bridges that gap using NLP, anomaly detection, and behavioral analytics.

Key Features
Doom Spending Detection — Identifies emotionally-driven purchases using a combination of time-pattern analysis (late-night transactions), Isolation Forest anomaly detection for spending spikes, rapid transaction burst detection, and NLP keyword sentiment on transaction notes.
Financial Health Score (0–100) — A composite wellness score built from four dimensions: savings ratio, spending consistency, doom spending penalty, and subscription overload index.
Subscription Leak Detector — Flags unused subscriptions (30+ days inactive), duplicate streaming services, and low-value recurring charges. Calculates exact monthly leakage in rupees.
Financial Persona Classifier — Categorizes users into behavioral profiles: Doom Spender, Subscription Addict, Impulse Buyer, or Smart Saver — using feature extraction across spending patterns.
Smart Budget Recommender — Generates personalized category-wise budget limits based on income using an adapted 50/30/20 framework, with overspending warnings and reduction targets.
Behavioral Pattern Analyzer — Detects peak spending hours, high-risk days of the week, and weekly spend trends. Surfaces human-readable insights like "26% of your spending happens after 10 PM."
